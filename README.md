Проект представляет из себя "Todo List".

Стек технологий:

- Java 17
- Spring
- Spring Boot
- Spring Data Rest
- H2database
- Hibernate
- Thymeleaf
- Liquibase
- Heroku

Набор реализованных функций:

- Создание задания.
- Изменение статуса задания (To do, Completed, Completed).
- Безвозвозвратное удаление задания из статуса "Completed" или "Deleted".
